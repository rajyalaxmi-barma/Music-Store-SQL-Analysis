# Music Store SQL Analysis

## 📖 Project Overview

This project analyzes a Music Store database using MySQL. The analysis answers business-related questions about customers, sales performance, invoices, artists, genres, and revenue using SQL queries.

---

## 🛠️ Tools & Technologies

- MySQL Workbench
- SQL

---

## 📂 Dataset

This project uses the **Music Store sample database** for SQL analysis. The dataset is publicly available and is commonly used for practicing SQL queries and data analysis.

---

## 📊 SQL Concepts Used

- SELECT Statements
- Filtering (WHERE)
- Sorting (ORDER BY)
- Aggregate Functions
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (RANK)

---

## 📈 Business Questions Solved

Some of the questions answered in this project include:

- Who is the senior-most employee?
- Which countries generate the highest revenue?
- Which city has the highest invoice total?
- Who are the top customers?
- Which music genre is most popular?
- Which artists generate the highest sales?
- Which customers spend the most in each country?

## ▶️ How to Run

1. Import the Music Store database into MySQL Workbench.
2. Open the SQL script.
3. Execute the queries one by one.
4. Review the results for each business question.
